About This Portfolio

This portfolio is based on a real software engineering project conducted using Marginum’s brain tumor detection device. The detector outputs only a numerical value representing tumor concentration. Using this raw data, we performed data-science-driven analysis to identify When a tumor signal is detected, Which tumor class is activated Based on this analysis, we implemented a real-time detection sound system and designed a GUI that provides immediate feedback to clinicians. The project incorporated SDLC principles, quality standards, and process methods learned throughout the course, making it a practical demonstration of applying software engineering discipline to a medical-support system.
